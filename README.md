# KITRO — Site e-commerce (React)

Réplique du design GAZU en React + Vite.

## Installation

```bash
npm install
```

## Lancer en développement

```bash
npm run dev
```

Puis ouvrir http://localhost:5173

## Build de production

```bash
npm run build
```

## Structure

- `src/components/Header.jsx` — barre du haut + navigation
- `src/components/Hero.jsx` — bannière "GAZU" avec le mannequin
- `src/components/CategoryStrip.jsx` — bloc MEN / WOMEN / KIDS
- `src/components/NewVibes.jsx` — section "NEW VIBES"
- `src/components/Features.jsx` — livraison / retours / qualité / paiement
- `src/components/BestOf.jsx` — grille produits "BEST OF GAZU"
- `src/components/Footer.jsx` — pied de page
